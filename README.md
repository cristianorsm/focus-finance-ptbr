# Focus Finance — Protótipo (pt-BR)

Protótipo navegável em **React + Tailwind (Vite)** com telas:
Início, Lançar gasto, Orçamento, Transações, Lembretes, Definir conta,
Recorrência e Revisar & Confirmar.

## Como rodar

```bash
npm install
npm run dev
```

Abra o endereço que o Vite informar (ex.: http://localhost:5173).

## Build de produção

```bash
npm run build
npm run preview
```

Tailwind está configurado no `tailwind.config.js` e estilos utilitários
customizados em `src/index.css` (camada `@layer components`).

---

> Observação: este é um **protótipo** sem backend/persistência.
